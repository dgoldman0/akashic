# Library applet package placeholder

Gate 1 reserves this directory for the future Library UI lens. It intentionally
contains no `library.f`, `library.uidl`, app descriptor, manifest, handler,
capability, or store.

When implemented in its later gate, this applet may depend on the Library
domain and portable TUI/interop clients. It must not import sibling applet or
sibling-domain internals. See
[`../../../../docs/tui/applets/library/library.md`](../../../../docs/tui/applets/library/library.md).
